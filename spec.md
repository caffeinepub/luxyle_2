# Specification

## Summary
**Goal:** Reorder sections on the Home page so that Featured Collections appears before the About section.

**Planned changes:**
- Move the `FeaturedCollections` component above the `AboutSection` component on the Home page, resulting in the order: Hero → Stats bar → FeaturedCollections → About → CollectionGallery → Reviews → FeedbackForm → AppointmentBooking.

**User-visible outcome:** When scrolling down the Home page, users will see the Featured Collections grid before the About Luxyle section.
